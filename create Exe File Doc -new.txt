1. pip install pyinstaller

2. go project where we want to create exe 
enter below command

"pyinstaller --name WebScrapper --collect-all customtkinter --icon=myicon.ico -w main.py"

#to avoid the error of babel number usebelow packge if we are using the tkcalender date
"pyinstaller --name AttendScan --collect-all customtkinter --icon=myicon.ico -w --hidden-import babel.numbers main.py"
"pyinstaller --name AttendScan --collect-all customtkinter --icon=myicon.ico -w --hidden-import babel.numbers main.py"

#below script is for console mode, to debug the errors
"pyinstaller --name AttendScan --collect-all customtkinter --icon=myicon.ico main.py"


		where WebScrapper is exe name which we want to give

		--collect-all customtkinter as we are using third party lib custome tkinter so we need to collect all its packages
			as it includes themes as well

		--icon=myicon.ico    --select our app logo as icon append to it

		main.py    select out main file from where scrept is gettigng started



3. After above step project is getting created in dist folder,              (we dont need build folder we can delete it)
	
	before ruunning the exe first copy our dir stucture to it
	like .env file, output folder , src folder ,a eevery other folder aaprt from build and dist

	#dont copy the build and dist folder
